## "스킬(Skills)"이란 무엇인가요?

**Claude Code**, **Gemini**, **Cursor**와 같은 AI 에이전트들은 똑똑하지만, 특정 도구나 작업 방식에 대한 구체적인 지식은 부족할 수 있습니다.
**스킬**은 AI에게 특정 작업을 매번 완벽하게 수행하는 방법을 가르치는 **전문화된 지침서(마크다운 파일)**입니다.

**비유하자면:** 여러분의 AI가 유능한 인턴이라면, **스킬**은 그들을 수석 엔지니어로 만들어주는 **표준 운영 절차(SOP)**와 같습니다.

## 빠른 시작: "스타터 팩(Starter Packs)"

저장소의 방대한 규모에 당황하지 마세요. 모든 것이 한꺼번에 필요하지는 않습니다.
즉시 실행해 볼 수 있도록 엄선된 **스타터 팩**을 준비했습니다.

전체 저장소를 한 번 설치(npx 또는 clone)한 후, 역할별(예: 웹 위저드, 해커 팩)로 필요한 스킬을 골라서 사용하면 됩니다.

### 1. 저장소 설치

**방법 A — npx (가장 쉬움):**

```bash
npx antigravity-awesome-skills
```

이 명령은 기본적으로 `~/.gemini/antigravity/skills`에 스킬을 복제합니다. 특정 도구용으로 설치하려면 `--cursor`, `--claude`, `--gemini`, `--codex`, `--kiro` 플래그를 사용하고, 특정 위치를 원하면 `--path <경로>`를 사용하세요.

**방법 B — git clone:**

```bash
# 범용 (대부분의 에이전트에서 작동)
git clone https://github.com/sickn33/antigravity-awesome-skills.git .agent/skills
```

### 2. 본인의 역할 선택

본인의 역할에 맞는 번들을 찾으세요 (상세 내용은 [bundles.md](../users/bundles.md) 참조):

| 역할 | 번들 이름 | 포함 내용 |
| :--- | :--- | :--- |
| **웹 개발자** | `Web Wizard` | React 패턴, Tailwind 숙달, 프론트엔드 디자인 |
| **보안 엔지니어** | `Hacker Pack` | OWASP, Metasploit, 모의해킹 방법론 |
| **매니저 / PM** | `Product Pack` | 브레인스토밍, 계획 수립, SEO, 전략 |
| **공통/필수** | `Essentials` | 클린 코드, 계획, 검증 (기본기) |

---

## 번들(Bundles) vs 워크플로우(Workflows)

번들과 워크플로우는 서로 다른 문제를 해결합니다:

- **번들** = 역할별로 엄선된 스킬 세트 (무엇을 고를 것인가).
- **워크플로우** = 단계별 실행 가이드 (어떻게 실행할 것인가).

## 스킬 사용 방법

설치가 완료되면 AI에게 자연스럽게 말을 걸면 됩니다.

### 예시 1: 기능 계획하기 (**Essentials**)

> "**@brainstorming**을 사용해서 새로운 로그인 흐름을 설계하는 걸 도와줘."

**결과:** AI가 브레인스토밍 스킬을 로드하고, 구조화된 질문을 던진 뒤 전문적인 사양서를 작성합니다.

### 예시 2: 코드 점검하기 (**Web Wizard**)

> "이 파일에 대해 **@lint-and-validate**를 실행하고 오류를 수정해 줘."

**결과:** AI가 스킬에 정의된 엄격한 린팅 규칙을 따라 코드를 깔끔하게 정리합니다.

## 자주 묻는 질문 (FAQ)

**Q: 모든 스킬을 다 설치해야 하나요?**
A: 전체 저장소를 한 번만 복제하면 됩니다. AI는 여러분이 호출한 스킬(또는 관련 있는 스킬)만 읽으므로 가볍게 유지됩니다.

**Q: 나만의 스킬을 만들 수 있나요?**
A: 네! **@skill-creator** 스킬을 사용하여 직접 만들어 보세요.

**Q: 무료인가요?**
A: 네. 원본 코드와 도구는 MIT 라이선스이며, 문서 콘텐츠는 CC BY 4.0 라이선스입니다.
